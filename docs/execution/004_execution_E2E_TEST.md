## 1. ユーザージャーニー

### UC4: 時間ベース実行（§1.1 #4）
| シナリオ ID | 前提 | 操作 | 期待結果 |
|---|---|---|---|
| UC4-S1 happy | セット（3 アイテム）あり | 開始 → アイテム1 終了 → 次へ → アイテム2 終了 → 一時停止 → 同じ活動再開 → 終了 → セット終了 | 各アイテムに経過時間記録、session done |
| UC4-S2 pause | 実行中 | 一時停止 → （時間経過）→ 再開 | 経過時間に pause 分が含まれない |
| UC4-S3 background | 実行中 | タブを背景化/時間進める → 復帰 | 経過時間が壁時計どおり（タイマー狂わない） |
| UC4-S4 restart | 実行中 | リロード | 進行中実行が復元される |

### UC5: 今日メモ（§1.1 #5）
| シナリオ ID | 操作 | 期待結果 |
|---|---|---|
| UC5-S1 | アイテム実行中にメモ入力 | record に保存・表示 |

### 達成記録
| シナリオ ID | 操作 | 期待結果 |
|---|---|---|
