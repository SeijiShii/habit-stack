# 改修: アカウント切替時の計時停止条件緩和 + デバイス⇔アカウント同期ポリシー

- **issue / slug**: R20260615-001 / account-switch-stop-sync
- **実施日**: 2026-06-15
- **対象機能**: ../README.md（_shared/auth）
- **基準 SPEC**: ../001__shared_auth_SPEC.md
- **改修要望**:
  1. `/account` 遷移での計時強制停止を緩和し、アカウント切替（Google ログイン / サインアウト）を明示的に行ったときを停止条件にする。
  2. 停止時、計時中の活動があれば「計時中の活動を停止しますか」と確認したうえで停止する。
  3. 強制停止時に作成済みセット・活動が保存されず消える不具合を是正。保存したうえで切替ポリシーに従い削除可否を決める。
  4. 未連携 Google アカウントにログイン → デバイスデータをアカウントへ連携・保存。
  5. 既にデータを持つ Google アカウントにログイン → そのデータでデバイスを上書き。
  6. サインアウト → デバイスのデータを削除。
- **状態**: 設計中
